# Блокчейн простыми словами

Разбираемся с технологией, которая меняет финансовый мир.

## Что такое блокчейн?

Блокчейн - это **цифровая книга записей**, которая:

- 📄 Хранится одновременно на множестве компьютеров
- 🔒 Защищена криптографией  
- ⛓️ Состоит из связанных блоков транзакций
- 🚫 Не может быть изменена задним числом

## Как это работает?

### Процесс добавления транзакции:

1. **Инициация** - пользователь создает транзакцию
2. **Проверка** - сеть валидирует операцию
3. **Включение в блок** - транзакция добавляется в новый блок
4. **Консенсус** - сеть согласовывает новый блок
5. **Добавление** - блок присоединяется к цепочке

```
[Блок 1] -> [Блок 2] -> [Блок 3] -> [Новый блок]
   |           |           |            |
Genesis    Транзакции  Транзакции   Ваша операция
```

## Применения в финансах

### 💰 Криптовалюты
- Bitcoin - первая цифровая валюта
- Ethereum - платформа смарт-контрактов
- Стейблкоины - привязанные к фиату

### 🏦 Банковские услуги
- Международные переводы без посредников
- Торговое финансирование
- Документооборот и верификация

### 📜 Смарт-контракты
Автоматические договоры, которые исполняются сами:

```solidity
contract SimplePayment {
    function pay() public payable {
        if (conditions_met) {
            recipient.transfer(amount);
        }
    }
}
```
